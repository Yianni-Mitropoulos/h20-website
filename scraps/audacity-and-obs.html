General Offline Use: Part 3.

Shut down dXX-offline-goff at this point.

Two more software packages that are probably worth having are Audacity (for audio editing) and OBS Studio (for recording yourself). Both are currently free and open source. However, each comes with its own set of security and privacy risks.

Audacity is currently owned by Muse Group. A telemetry incident in 2021 caused extensive community backlash, and brought into question whether Muse Group respects the privacy of its users.

OBS is developed and maintained by the OBS Project, which is community-driven. However, the project has corporate sponsors, including both Logitech and YouTube.

For the sake of safety, I personally recommend duplicating dXX-offline-goff, and calling the result dXX-offline-goff-plus. Inside this duplicate, we can install Audacity and OBS. This is intended to help control the blast radius, just in case either one of them begins misbehaving in the not-too-distant future.

You can open a terminal in dXX-offline-goff-plus like so: 

# ==================== # You can use SHIFT+INSERT to paste into
#     dom0 > xterm     # xterm, or use the middle mouse button.
# ==================== #

# Open a terminal as the root user
qvm-run -u root dXX-offline-goff-plus xterm

And then run the following code to install Audacity and OBS Studio. 

# ===================================== # You can use SHIFT+INSERT to paste into
#     dXX-offline-goff-plus > xterm     # xterm, or use the middle mouse button.
# ===================================== #

# Update package list
until sudo apt update; do sleep 1; done

# Audacity
# Owned by Muse Group
# Telemetry incident in 2021 caused extensive community backlash
until sudo apt install audacity; do sleep 1; done

# OBS Studio
# Community-driven (but with some corporate sponsors, e.g., Logitech, YouTube)
# Currently in high regard amongst many FOSS advocates.
# BUT: May face corporate pressure moving forwards.
until sudo apt install obs-studio; do sleep 1; done
